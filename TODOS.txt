Coisas para melhorar:

1. Remover console.log() excessivos 
e usar os outrso métodos da classe console {console.error(); console.warning(); ...}

2. Tentar criar fuções que podem ser reutilizadaso no código 
Exemplo: A funcção de validação de campos na controller.

3. Implementar a validação de e-mail.

Google Calendar:

1. Ver o esquema do embed da agenda para o front.

2. Criar uma conta de serviço para a API do Calendar.

3. Ver como pegar o Id de um evento para implementar os outros métodos (*update, get e delete).
*: Talvez a gente não implemente o update de agendamento, apenas deletamos o antigo e cadastramos um novo.
